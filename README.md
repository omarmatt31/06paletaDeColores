## 🚀 Cómo comenzar

1. Clona el repositorio:
```bash
git clone https://github.com/omarmatt31/06paletaDeColores.git
```

2. Instala dependencias:
npm install
```

3. Inicia la aplicación:
```bash
npm run dev
```
